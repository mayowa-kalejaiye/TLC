# The Light Community - Brand Colors

Based on "The Rooted Clan" logo design.

## Primary Color Palette

### Dark Green (Primary Background)
- **Main Green:** `#0D3B3B` - Primary background color
- **Dark Green:** `#082828` - Darker shade for depth
- **Light Green:** `#1A5555` - Lighter shade for hover/accents

**Usage:** Footer, Give section, dark sections, backgrounds

### Gold/Yellow (Primary Accent)
- **Main Gold:** `#F5B82E` - Primary accent color (from logo)
- **Dark Gold:** `#D9A325` - Hover states, darker accents
- **Light Gold:** `#FFCA5A` - Highlights, light accents

**Usage:** CTAs, buttons, icons, highlights, text accents

### Neutral Colors
- **Cream:** `#FFF9F0` - Light backgrounds
- **Cream Dark:** `#FFF3E0` - Slightly darker cream
- **Navy:** `#1A202C` - Text color (dark)
- **Navy Light:** `#2D3748` - Secondary text

## Color Combinations

### Primary Combo (High Impact)
- Background: Dark Green `#0D3B3B`
- Foreground: Gold `#F5B82E`
- Text: White

### Light Combo (Clean, Welcoming)
- Background: Cream `#FFF9F0`
- Foreground: Gold `#F5B82E`
- Text: Navy `#1A202C`

### Dark Combo (Bold, Authoritative)
- Background: Navy `#1A202C`
- Foreground: Gold `#F5B82E`
- Text: White

## Logo Colors

From "The Rooted Clan" logo:
- Figure & Roots: Gold `#F5B82E`
- Background: Dark Green `#0D3B3B`
- Text: Gold `#F5B82E`

## Typography Pairing

- **Headings:** Playfair Display (serif) in Navy or White
- **Body:** Inter (sans-serif) in Navy/Navy-light or White
- **Accents:** Gold for emphasis

## Accessibility

All color combinations meet WCAG AA standards:
- ✅ Gold on Dark Green (4.5:1+)
- ✅ White on Dark Green (15:1+)
- ✅ Navy on Cream (12:1+)
- ✅ Gold on Navy (4.5:1+)

## Tailwind CSS Classes

```css
/* Backgrounds */
.bg-tlc-green        /* #0D3B3B */
.bg-tlc-green-dark   /* #082828 */
.bg-tlc-green-light  /* #1A5555 */
.bg-tlc-gold         /* #F5B82E */
.bg-tlc-cream        /* #FFF9F0 */
.bg-tlc-navy         /* #1A202C */

/* Text */
.text-tlc-green
.text-tlc-gold
.text-tlc-navy

/* Borders */
.border-tlc-green
.border-tlc-gold
```

## Where Each Color is Used

### Dark Green (`#0D3B3B`)
- Footer
- Give section background
- Alternative dark sections

### Gold (`#F5B82E`)
- All CTA buttons
- Icons and highlights
- Navigation "Give Now" button
- Section labels (uppercase headings)
- Link hover states
- Newsletter subscribe button

### Cream (`#FFF9F0`)
- Welcome section background
- Devotionals section background
- Newsletter section background
- Card backgrounds

### Navy (`#1A202C`)
- Body text
- Headings on light backgrounds
- Navigation text

## Semantic Usage

- **Action:** Gold (buttons, links)
- **Trust:** Dark Green (footer, giving)
- **Welcome:** Cream (warm sections)
- **Authority:** Navy (text, headings)
