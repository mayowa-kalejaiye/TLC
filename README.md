# The Light Community Church - Official Website

**Souls saved. Men Trained. Nations taken.**

Modern, interactive church website built with Next.js 14, TypeScript, and Tailwind CSS.

## 🚀 Features

- **Responsive Design** - Mobile-first approach optimized for Nigerian audiences
- **Sermon Archive** - YouTube integration with searchable sermon library
- **Daily Devotionals** - Fresh spiritual content every day
- **Ministry Hub** - Information and signup for all departments
- **Online Giving** - Secure donations via Paystack/Flutterwave
- **Events Management** - RSVP and registration for church programs
- **Testimonies** - Share and read life transformation stories
- **Newsletter** - Email subscription for updates and devotionals

## 🛠️ Tech Stack

- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Animations:** Framer Motion
- **Forms:** React Hook Form + Zod validation

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
